Abstract - With the rapid development of the Android platform, numerous applications (apps) aredeveloped to provide new or enhanced services using resources on mobile devices. On one hand,these apps can bring convenience for users in context which has little privacy risk. On the otherhand, misuse of these resources by apps may result in privacy leakage in context which has strictrequirements for privacy. Therefore, fine-grained resource usage constraint mechanisms that takethe context information into account are in demand. This paper presents an enhanced context-aware resource usage control system (EasyPrivacy) in Android to provide adaptive resource usagecontrol. The main contribution of this paper is -
1. A Context-aware Resource Usage Control System - EasyPrivacy
2. Policy Conflict Consideration
